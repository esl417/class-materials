# Class materials

Student materials for the classes. Each class has its own folder. You don't need to read
the code — your instructor will tell you which folder to use, and Claude does the rest.

## Classes

- **`class-2-dashboard/`** — building a dashboard from your website's analytics.
  - `ga4-student-kit/` — connects Claude to your Google Analytics (read-only). Start with
    its `START_HERE.md`.

## How students use this

You generally won't clone this by hand. Your class deck gives you a single message to
paste into **Claude Code** — it tells Claude which folder to use and what to do. Claude
clones this repo and runs the setup for you.
